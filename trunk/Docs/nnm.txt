RealChat был написан как альтернатива популярному Intranet Chat'у (автор 
Александр Ворожун) для локальных сетей. При всех достоинствах iChat'а у него 
были серьезные недостатки: слабо развита серверная часть, нестабильность при 
большом количестве пользователей, уязвимость используемого протокола передачи
сообщений.. Автор в какой-то момент прекратил работу над чатом и не стал 
передавать исходники в общее пользование для развития проекта.

RealChat использует широко известный протокол передачи сообщений IRC, имеет 
внешний вид и функции как у iChat и обладает некоторыми уникальными 
особенностями.


[b]Обычные возможности[/b]
- Поддержка протокола IRC - каналы, приватные сообщения, записки, DCC CHAT,
  DCC SEND, DCC RESTORE.
- Список каналов с фильтром, поиском и сортировкой
- Список быстрых сообщений
- Сборщик заметок по ключевым словам
- Смена шрифтов
- Звуки на события
- Настаиваемые шаблоны сообщений
- Список принятых и отправленых файлов
- Цветной, наклонный, подчеркнутый, жирный текст
- Всплывающие окна для приватных сообщений
- Записки из трея (значок рядом с часами)
- Список игнорируемых пользователей
- Выделение гиперссылок на http://
- Группировка пользователей
- Подключение через HTTP/SOCKS прокси

[b]Особые возможности[/b]
- Анимированные смайлы
- Доска обьявлений
- Шифрование сообщений на канале и в приватах (RC4 + BASE64 для избавления от
  непечатных символов).
- Отображение аватаров пользователей
- Плагины в стиле командной строки
- Отсоединение окон чата от окна основной программы

[b]Как пользоваться[/b]

Для начала, зайдите в настройки (Options) и на странице Подключение 
(Connection) задайте имя пользователя, адрес и порт сервера. Для 
подключения/отключения к серверу жмите кнопку Подключить (Connect).

По кнопке Список каналов (Channels list) можно получить список каналов на 
сервере.

Правый щелчок мышью на элементах чата вызывает контекстное меню.

Можно задать список команд, выполняемых при подключении к серверу. Это могут 
быть команды смены ника, подключения к каналам, идентификации ника, 
приветствия, итд..

Шифрование сообщений включается командой /PASS [пароль]
Шифрование отключается, если пароль не задан.

Можно создавать группы ников. Для помещения ника в группу перетащите его 
мышкой при нажатом SHIFT.
